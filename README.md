\# Echoes of Tomorrow



Echoes of Tomorrow is a 2D Unity platformer game featuring time manipulation, puzzle elements, and story-driven gameplay.



\## Features

\- 2D character movement and animations  

\- Multiple levels with increasing difficulty  

\- Platforms, obstacles, and rising platforms  

\- Custom character sprites  

\- Camera follow and zoom mechanics  

\- Unique world art and environment  



\## Technologies Used

\- Unity Engine  

\- C# Scripts  

\- 2D Sprite Artwork  

\- Animation Controllers  



\## Project Structure

\- \*\*Assets/\*\*: Art, scenes, scripts, prefabs  

\- \*\*ProjectSettings/\*\*: Unity project settings  

\- \*\*Packages/\*\*: Package manager dependencies  



\## How to Run

1\. Open the project in Unity Hub  

2\. Select the Unity version used in the project  

3\. Click \*Open\* and run the scene `SampleScene.unity`
